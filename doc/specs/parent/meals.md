# 餐費紀錄

**路徑**: `/parent/meals`
**角色**: Parent

## 核心目的

查看孩子的餐費紀錄。

## MVP 功能

### 餐費列表

- 按月份分組顯示，每月一個區塊
- 每月區塊顯示：月份標題、當月總金額、該月所有餐費明細
- 每筆明細顯示：日期、餐別（早/午/晚/點心）、金額、備註（若有）

### 篩選

- 近 3 個月（預設）
- 近 6 個月
- 指定月份範圍

### 匯出功能

- 匯出按鈕位於頁面頂部右側
- 支援匯出 CSV 格式
- 匯出範圍依目前篩選條件
- 匯出欄位：日期、餐別、金額、備註

### 空狀態

若無餐費紀錄，顯示空狀態提示
