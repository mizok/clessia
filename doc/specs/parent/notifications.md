# 通知中心（家長）

**路徑**: `/parent/notifications`
**角色**: Parent

## 核心目的

查看課務異動通知。

## MVP 功能

### 通知列表

- 以時間軸列表呈現，最新在上
- 每則通知顯示：
  - 通知類型圖示（調課/代課/停課用不同圖示）
  - 標題（例如：「國一數學 A 班 調課通知」）
  - 摘要內容（例如：「1/30 的課堂已調整至 2/2」）
  - 發送時間（相對時間）
  - 已讀/未讀狀態（未讀有藍點標記）

### 點擊展開

原地展開，不另開 Popup：
- 異動類型與原因
- 原課堂資訊（日期、時間、老師）
- 新課堂資訊（若為調課/代課）
- 相關課堂連結

### 操作

- 點擊即標記為已讀
- 「全部標為已讀」按鈕
- 通知保留 90 天

## 資料依賴

| 操作 | 資料表 |
|------|--------|
| 讀取 | `notifications` |
| 寫入 | `notifications`（更新已讀狀態） |
