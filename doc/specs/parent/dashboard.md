# 家長儀表板

**路徑**: `/parent/dashboard`
**角色**: Parent

## 核心目的

家長首頁，快速掌握孩子今日狀況。

## MVP 功能

### 孩子切換器

- 若有多個孩子，可切換查看
- 預設顯示第一個孩子

### 今日到班狀態

- 孩子今天是否已到班
- 若今日無課則顯示「今日無課」

### 今日/近期課表

- 以卡片 Grid 呈現
- 今天有課就顯示今日課表，若無則顯示最近 3 天的課表
- 每張卡片顯示時間、課程名、老師、分校
- 課務異動標記（調課/代課/停課）
- 有小考標籤
- 點擊卡片以 Popup 展開課堂詳情

### 推薦加選課程

- 顯示 2-3 個推薦課程卡片
- 每張卡片顯示：課程名稱、上課時間摘要、費用
- 「查看更多課程」按鈕
- 點擊課程卡片開啟詳情 Popup

### 最新聯絡簿

- 輪播卡片呈現各課程的最新聯絡簿
- 每張卡片顯示課程名稱與摘要
- 點擊展開完整內容

### 考試與成績

- 以卡片 Grid 呈現
- 時間範圍：未來 7 天內的考試 + 過去 7 天內有成績的考試
- 卡片內容：考試名稱、日期、分數（若有）、狀態標籤
- 卡片樣式：
  - 今天考試：暖銅橘淡底
  - 未來考試：淡米白底
  - 滿分成績：淡金底 + 鼓勵語
  - 及格成績：淡綠底
  - 不及格成績：淡紫底 + 鼓勵語
- 新成績加「NEW」標籤

### 待繳費提醒

若有待付款的繳費單：
- 顯示提醒卡片
- 即將到期（3 天內）：卡片邊框變紅色
- 已過期：卡片背景變淡紅色

### 續課預告提醒

在 T-5 ~ T-0 期間顯示：
- 標題：「[期間] 續課預告」
- 「申請異動」按鈕
