# 學生資料

**路徑**: `/admin/students`
**角色**: Admin
**分組**: 學務管理

## 核心目的

管理學生基本資料。

## MVP 功能

### 學生列表

- 篩選：姓名搜尋、年級、分校（依報名的開課班所屬分校）
- 每筆顯示：姓名、年級、學校、關聯家長、報名課程數、狀態
- 點擊進入編輯或查看詳情

### 新增/編輯學生 Popup

| 欄位 | 類型 | 必填 | 說明 |
|------|------|------|------|
| 姓名 | 文字 | 是 | |
| 年級 | 下拉選單 | 是 | K, P1-P6, J1-J3, S1-S3 |
| 就讀學校 | 文字 | 是 | |
| 生日 | 日期 | 否 | |
| 性別 | 單選 | 否 | 男/女/不透露 |
| 電話 | 文字 | 否 | 學生本人電話 |
| 地址 | 文字 | 否 | |
| 緊急聯絡人 | 文字 | 否 | 姓名+電話 |
| 備註 | 長文字 | 否 | |

### 學生詳情頁

- 基本資料（可編輯）
- 關聯家長列表
- 報名的開課班列表（連結到學生報名頁面）
- 出勤統計摘要
- 成績統計摘要

## 資料依賴

| 操作 | 資料表 |
|------|--------|
| 讀取 | `students`, `parents`, `parent_student_relations`, `enrollments`, `attendances`, `grades` |
| 寫入 | `students` |
