# 出勤紀錄

**路徑**: `/admin/attendance`
**角色**: Admin
**分組**: 學務管理

## 核心目的

查看和修正學生出勤狀態。

## MVP 功能

### 篩選列

- 日期範圍
- 課程
- 開課班
- 出勤狀態（全部/有異常）

### 課堂列表

- 按日期排序（最近在上）
- 每筆顯示：日期、時間、開課班名稱、老師、出勤摘要（例如「10 出席 / 1 遲到 / 1 缺席」）
- 有異常的課堂（有缺席或遲到）加標記提醒
- 點擊開啟管理員課堂詳情 Popup

### 快速操作

- 在列表中可快速展開課堂的學生出勤狀態
- 直接在展開區域修改出勤狀態並儲存

### 出勤狀態

- 出席（present）
- 遲到（late）
- 請假（leave）
- 缺席（absent）

## 資料依賴

| 操作 | 資料表 |
|------|--------|
| 讀取 | `sessions`, `attendances`, `students`, `classes` |
| 寫入 | `attendances` |
