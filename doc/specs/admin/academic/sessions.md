# 課堂搜尋

**路徑**: `/admin/sessions`
**角色**: Admin
**分組**: 教務管理

## 核心目的

依條件搜尋課堂，快速找到特定課堂。

## MVP 功能

### 篩選條件

- 日期範圍（必填）
- 課程（可選）
- 開課班（可選，依課程連動）
- 老師（可選）
- 分校（可選）
- 課堂狀態：全部/正常/已停課/有異動

### 快速篩選標籤

- 「今天」
- 「本週」
- 「有異動」
- 「已停課」

### 搜尋結果

- 課堂列表，按日期排序
- 每筆顯示：日期、時間、開課班、老師、分校、狀態標記
- 點擊開啟課堂詳情 Popup

## 資料依賴

| 操作 | 資料表 |
|------|--------|
| 讀取 | `sessions`, `classes`, `teachers`, `campuses`, `schedule_changes` |
