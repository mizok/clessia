# 課程列表

**路徑**: `/admin/courses`
**角色**: Admin
**分組**: 教務管理

## 核心目的

管理補習班開設的課程（教學內容分類）。

## MVP 功能

### 課程列表

- 每筆顯示：課程名稱、科目、開課班數量、狀態
- 點擊進入編輯

### 新增/編輯課程 Popup

| 欄位 | 類型 | 必填 | 說明 |
|------|------|------|------|
| 課程名稱 | 文字 | 是 | 例如：國一數學、國二英文 |
| 科目 | 下拉選單 | 是 | 國文/英文/數學/自然/社會/其他 |
| 說明 | 長文字 | 否 | 課程簡介 |
| 狀態 | 開關 | 是 | 啟用/停用 |

### 操作

- 新增課程
- 編輯課程
- 啟用/停用課程

## 資料依賴

| 操作 | 資料表 |
|------|--------|
| 讀取 | `courses`, `classes` |
| 寫入 | `courses` |
