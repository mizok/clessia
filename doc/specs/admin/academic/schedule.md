# 排課管理

**路徑**: `/admin/schedule`
**角色**: Admin
**分組**: 教務管理

## 核心目的

管理開課班的上課時間規則。

## MVP 功能

### 主要功能

- 依開課班檢視上課時段
- 編輯時段（時間、老師、生效日期）
- 檢查時段衝突（同老師、同教室）

### 時段列表

- 篩選：分校、課程、開課班、老師
- 每筆顯示：開課班、星期、時間、老師、教室、生效期間
- 衝突標記（紅色警示）

### 衝突檢查

- 同一老師同一時段不可重複
- 同一教室同一時段不可重複
- 衝突時顯示警告訊息

## 資料依賴

| 操作 | 資料表 |
|------|--------|
| 讀取 | `schedules`, `classes`, `teachers`, `classrooms` |
| 寫入 | `schedules` |
