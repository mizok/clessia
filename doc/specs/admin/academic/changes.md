# 課務異動

**路徑**: `/admin/changes`
**角色**: Admin
**分組**: 教務管理

## 核心目的

查看調課/代課/停課的歷史紀錄。

## MVP 功能

### 篩選列

- 日期範圍
- 異動類型（全部/調課/代課/停課）
- 課程
- 老師

### 異動列表

按建立時間排序（最新在上），每筆顯示：

- 異動類型圖示（調課/代課/停課用不同圖示）
- 課堂資訊：開課班、原日期時間
- 異動內容摘要：
  - 調課：「調整至 [新日期時間]」
  - 代課：「由 [代課老師] 代課」
  - 停課：「已停課」
- 原因（若有）
- 建立時間、建立者

### 詳情展開內容

- 原課堂完整資訊
- 異動後資訊（若為調課/代課）
- 通知發送狀態：已通知的家長/老師數量
- 操作紀錄：誰在什麼時候建立的

## 資料依賴

| 操作 | 資料表 |
|------|--------|
| 讀取 | `schedule_changes`, `sessions`, `notifications` |
