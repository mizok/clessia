# 分校設定

**路徑**: `/admin/campuses`
**角色**: Admin
**分組**: 系統設定

## 核心目的

管理分校資訊與教室。

## MVP 功能

### 分校列表

- 每筆顯示：分校名稱、地址、聯絡電話、狀態（啟用/停用）
- 點擊進入編輯

### 新增/編輯分校 Popup

| 欄位 | 類型 | 必填 | 說明 |
|------|------|------|------|
| 分校名稱 | 文字 | 是 | 例如：台北校、台中校 |
| 地址 | 文字 | 否 | |
| 聯絡電話 | 文字 | 否 | |
| 狀態 | 開關 | 是 | 啟用/停用 |

### 銀行帳戶資訊

- 戶名
- 銀行名稱
- 帳號

### 出勤模式設定

| 模式 | 說明 |
|------|------|
| 日到班模式（預設） | 學生打卡到班後，系統自動推算當天所有課堂的出席狀態 |
| 課堂模式 | 每堂課獨立計算出勤，需管理員或任課老師確認 |

### 教室管理（分校子頁籤）

- 教室列表
- 新增/編輯教室
- 教室欄位：名稱、容納人數、設備（可選）

## 資料依賴

| 操作 | 資料表 |
|------|--------|
| 讀取 | `campuses`, `classrooms` |
| 寫入 | `campuses`, `classrooms` |
