# 費用方案管理

**路徑**: `/admin/fee-templates`
**角色**: Admin
**分組**: 財務管理

## 核心目的

定義開課班的收費標準。

## MVP 功能

### 費用方案列表

- 篩選：開課班、生效期間
- 每筆顯示：方案名稱、關聯開課班、金額、生效期間、狀態

### 新增/編輯費用方案 Popup

| 欄位 | 類型 | 必填 | 說明 |
|------|------|------|------|
| 方案名稱 | 文字 | 是 | 如「國一數學 A 班 - 2026 春季」 |
| 關聯開課班 | 下拉選單 | 是 | 選擇開課班 |
| 金額 | 數字 | 是 | 收費金額 |
| 生效起始日 | 日期 | 是 | |
| 生效結束日 | 日期 | 否 | 留空表示持續有效 |
| 按比例計算 | 開關 | 否 | 學期中加入是否按比例計算 |
| 說明 | 長文字 | 否 | 如「含教材費」 |
