# 報名審核

**路徑**: `/admin/enrollment-requests`
**角色**: Admin
**分組**: 報名管理

## 核心目的

審核家長提交的報名申請。

## MVP 功能

### 申請列表

- 篩選：狀態、日期、來源（public_form / parent_portal / add_course / renewal）
- 狀態標籤：
  - 待審核（灰色）
  - 待付款（暖銅橘）
  - 已完成（綠色）
  - 已拒絕（紅色）
  - 已取消（淺灰）
  - 已過期（淺灰）
- 每筆顯示：申請日期、家長姓名、學生姓名、開課班、來源、狀態

### 審核操作

**審核通過**：
1. 查看申請詳情
2. 確認費用（可調整金額，如早鳥優惠、兄弟姊妹折扣）
3. 建立繳費單
4. 系統同步建立 `pending_payment` 狀態的 Enrollment

**拒絕**：
1. 填寫拒絕原因
2. 確認拒絕

### 申請詳情

- 家長資訊
- 學生資訊
- 報名的開課班
- 期望開始日期
- 備註
- 狀態時間軸

## 資料依賴

| 操作 | 資料表 |
|------|--------|
| 讀取 | `enrollment_requests`, `classes`, `students`, `fee_templates` |
| 寫入 | `enrollment_requests`, `enrollments`, `invoices`, `tasks` |
