# 試聽管理

**路徑**: `/admin/trials`
**角色**: Admin
**分組**: 報名管理

## 核心目的

管理試聽申請、安排試聽、跟進結果。

## MVP 功能

### 申請列表

- 篩選：狀態、日期、課程
- 狀態標籤：
  - 待處理（灰色）
  - 已安排（藍色）
  - 已完成（綠色）
  - 已取消（淺灰）
- 每筆顯示：申請日期、家長姓名、學生姓名、想試聽的課程、狀態

### 安排試聽 Popup

- 選擇試聽課堂（從可選課堂中選擇）
- 填寫聯繫備註
- 確認安排

### 試聽結果記錄

- 家長意願：要報名/不報名/考慮中
- 不報名原因（若選擇不報名）
- 備註

### 狀態流轉

```
待處理 → 已安排 → 已完成
    ↓
  已取消
```

## 資料依賴

|------|--------|

| 寫入 | `trial_requests`, `tasks` |
