# 待處理工作

**路徑**: `/admin/tasks`
**角色**: Admin

## 核心目的

集中管理所有待辦任務，支援認領與處理。

## MVP 功能

### 任務列表

- 依優先級/截止日排序
- 篩選功能：
  - 任務類型（試聽安排、報名審核、繳費確認、試聽跟進、續課異動處理、年級不符檢查）
  - 狀態（待認領、處理中、已完成、已取消）
  - 認領者

### 任務操作

- 認領任務
- 轉派任務（給同團隊其他管理員）
- 點擊進入關聯頁面處理

### 任務卡片顯示

- 任務類型圖示
- 關聯對象摘要
- 截止日期（即將到期以紅色標示）
- 優先級標籤
- 認領者（若已認領）

## 資料依賴

| 操作 | 資料表 |
|------|--------|
| 讀取 | `tasks`, `enrollment_requests`, `invoices`, `trial_requests` |
| 寫入 | `tasks` |
