# 課程日曆

**路徑**: `/admin/calendar`
**角色**: Admin

## 核心目的

以日曆視圖查看全校課堂，快速進入課堂詳情。

## MVP 功能

### 視圖設計

- **桌面版**：左側小型月曆 + 右側課堂列表
- **手機版**：頂部月曆選擇器（可收合）+ 課堂列表

### 篩選列

- 課程篩選（下拉選單）
- 開課班篩選（依課程連動）
- 老師篩選

### 月曆區塊

- 小型月曆，有課的日期顯示圓點標記
- 點擊日期可快速跳轉至該日課堂
- 月份切換（< 2月 >）和「今天」按鈕

### 課堂列表區塊

- 按日期分組顯示課堂，預設顯示本週
- 每筆課堂顯示：時間、開課班名稱、老師名稱
- 課務異動標記（調課/代課/停課）
- 點擊開啟課堂詳情 Popup

### 課堂詳情 Popup（管理員版）

- 查看學生名單和出勤狀態
- 修改出勤狀態（不限時間）
- 建立請假
- 執行調課/代課/停課
- 查看聯絡簿（唯讀）

## 資料依賴

| 操作 | 資料表 |
|------|--------|
| 讀取 | `sessions`, `classes`, `teachers`, `schedule_changes`, `attendances`, `teacher_logs` |
| 寫入 | `attendances`, `leaves`, `schedule_changes` |
