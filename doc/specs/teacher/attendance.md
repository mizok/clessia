# 點名

**路徑**: `/teacher/attendance`
**角色**: Teacher

## 核心目的

查看和修改當天課堂的出勤狀態。

## MVP 功能

### 今日課堂列表

- 顯示今天自己任課的所有課堂
- 每筆顯示：時間、開課班名稱、出勤摘要

### 學生名單與出勤狀態

- 展開課堂後顯示學生列表
- 每位學生顯示：姓名、目前出勤狀態

### 修改出勤

- 僅限當天課堂
- 點擊學生可切換狀態：出席 → 遲到 → 請假 → 缺席

### 快速標記

- 全部出席：一鍵將所有學生標記為出席
- 批次遲到：選擇多位學生後批次標記為遲到

## 資料依賴

| 操作 | 資料表 |
|------|--------|
| 讀取 | `sessions`, `students`, `enrollments`, `attendances` |
| 寫入 | `attendances`（僅當天） |

- 非當天課堂僅可查看，不可修改
