# 課表（我的課表）

**路徑**: `/teacher/schedule`
**角色**: Teacher

## 核心目的

查看自己任課的課堂，進入課堂詳情。

## MVP 功能

### 視圖設計

- **桌面版**：左側小型月曆 + 右側課堂列表
- **手機版**：頂部月曆選擇器（可收合）+ 課堂列表

### 月曆區塊

- 小型月曆，有課的日期顯示圓點標記
- 點擊日期可快速跳轉至該日課堂
- 月份切換（< 2月 >）和「今天」按鈕

### 課堂列表區塊

- 按日期分組顯示課堂，預設顯示本週
- 每個日期標題顯示：日期、星期（例如「2/5（三）」）
- 每筆課堂顯示：時間、開課班名稱、標記

### 課堂卡片標記

- **待處理標記（●）**：暖銅橘色小圓點
  - 觸發條件：過去課堂尚未填寫聯絡簿，或有關聯考試但成績尚未登錄
- **[有小考]**：該課堂有關聯考試
- **[代課]**：自己代課（非原任課老師）
- **課務異動標記**：調課/停課等

### 圖例說明

- ● 待處理：聯絡簿未填或成績未登錄
- [有小考] 該課堂有安排考試
- [代課] 代課課堂
- 灰底 = 已停課

### 課堂詳情 Popup

點擊課堂開啟：

- 基本資訊（開課班、日期時間、分校）
- 學生名單與出勤狀態
- 修改出勤（僅當天）
- 編輯聯絡簿按鈕 → 開啟聯絡簿編輯 Popup
- 查看/建立小考

## 資料依賴

| 操作 | 資料表 |
|------|--------|
| 讀取 | `sessions`, `classes`, `teacher_logs`, `assessments`, `grades`, `schedule_changes`, `attendances`, `students` |
| 寫入 | `attendances`（僅當天）, `teacher_logs`, `assessments` |
